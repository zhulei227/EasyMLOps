文本特征提取¶

主要三类特征

  • bow/tfidf等常用特征

  • lsi/lda等主题模型特征

这些pipe模块的前置模块即是前面的文本处理结果，注意：处理好的token要求空格分割

相似文本检索¶

相似文本检索通常有两类方式：

  • 一类是通过字符串匹配的方式

  • 另一类则首先抽取文本特征向量，然后通过计算向量相似度的方式获取到相似文本

我们这里也有两种实现，基于ElasticSearch的文本检索，以及基于Faiss的特征向量检索两种

基于Faiss的特征向量检索¶

  • faiss具有不同的索引构造方式，以及不同的相似度度量方式，这部分可以通过create_index_param和create_index_measure设置，各种方式的优缺点以及适合场景还请访问官方文档:https://github.com/facebookresearch/faiss

  • 另外一个重要的参数是 add_production_data：是否要为生产数据也创建缩影，默认False

  • 由于index以及raw_data都比较耗费空间，这俩数据需要通过save_index_raw_data和load_index_raw_data接口单独手动保存或加载，通过pipeline的save和load只会保存/加载除index和raw_data以外的其余参数

该模块需要pip install faiss-cpu或pip install faiss-gpu，如下，构造了一个128维Lsi主题向量拼接一个128维的word2vec向量构成的特征向量，做相似文本检索

import pandas as pd
data=pd.read_csv("./data/demo2.csv",encoding="gbk")

x_train=data[:5000]
x_test=data[5000:]
y_train=x_train["label"]
y_test=x_test["label"]
del x_train["label"]
del x_test["label"]

from easymlops import NLPPipeline
from easymlops.table.preprocessing import *
from easymlops.table.ensemble import *
from easymlops.nlp.preprocessing import *
from easymlops.nlp.representation import *
from easymlops.nlp.similarity import FaissSimilarity

nlp=NLPPipeline()
nlp.pipe(FixInput())\
   .pipe(FillNa())\
   .pipe(SelectCols(cols=["review"]))\
   .pipe(ExtractChineseWords())\
   .pipe(ExtractJieBaWords())\
   .pipe(Parallel([LsiTopicModel(num_topics=128),Word2VecModel(embedding_size=128)]))\
   .pipe(Normalizer())\
   .pipe(FaissSimilarity(raw_data=x_train["review"].values.tolist(),create_index_param="IVF100,PQ16",create_index_measure="METRIC_INNER_PRODUCT"))

nlp.fit(x_train)
